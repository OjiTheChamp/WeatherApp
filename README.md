# 🌤️ BlockyWeather v3

A Roblox-style pixel art weather app built with vanilla HTML/CSS/JS.

**Live demo:** [ojithechamp.github.io/WeatherApp](https://ojithechamp.github.io/WeatherApp)

## Features
- 🗺️ Search any city worldwide
- 📡 Averages data from 9 nearby weather stations (Open-Meteo API)
- 🧪 Full air quality breakdown (PM2.5, PM10, Ozone, NO₂, SO₂, CO)
- ⏰ Live local time for any city (12-hour clock)
- 📅 24-hour hourly + 7-day daily forecast
- 🎮 Blocky Roblox-style pixel world background
- 👾 Builder Man, Cool Kid & Guest 666 walk across the scene
- 🕺 Michael Jackson moonwalks on full moons (whole town goes greyscale!)
- 🎨 10 background themes to choose from
- 🌧️ Real rain/snow particles when it's actually raining or snowing
- No API key needed — completely free

## How to use
Just open `index.html` in any browser. No server needed.

## Deploy to GitHub Pages
1. Upload `index.html` and `README.md` to a repo called `WeatherApp`
2. Go to Settings → Pages → Source: main branch → Save
3. Visit `https://ojithechamp.github.io/WeatherApp/`

## Built with
- [Open-Meteo](https://open-meteo.com/) — free weather API
- [Open-Meteo Air Quality](https://open-meteo.com/en/docs/air-quality-api) — free AQI API
- Vanilla HTML, CSS, JavaScript — no frameworks
